Frequenctly Asked Questions

Note: Here are some of the FAQs.

I already bought a parking permit and would like to avail the service as a rider. Can I return it?
Yes, parking pass can be returned and you will also receive a pro-rated refund based on the number of weeks remaining on the pass.
Is registration needed to be able to avail the ride sharing service?
Yes, a user should register first to be able to book a shared ride.
When will NSCC ride sharing be available for students?
An ongoing project has already been started and its timeline will soon be posted. Beta version of the web application is currently ongoing.
